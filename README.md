# The simple terminal

## Unique features (using dmenu)
+ **follow urls**  `alt-o`

## Bindings for
+ **scrollback** with `ctrl-↑/↓` or  scrolling the mouse
+ **zoom/change font size**: ctrl--, ctrl-+, ctr-0
+ **copy text** with `ctrl-c`, **paste** is `ctrl-v`

## Installation
```
git clone https://github.com/LukeSmithxyz/st
cd st
sudo make install
```
